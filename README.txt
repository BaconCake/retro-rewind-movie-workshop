===========================================================
  RETRO REWIND — MOVIE WORKSHOP
  Custom VHS covers & New Releases for Retro Rewind
===========================================================

QUICK START
-----------
1. Extract this folder anywhere on your PC.
2. Double-click RR_Movie_Workshop.exe
3. On first run, the setup wizard will ask you to locate:
   - Your Retro Rewind game folder (auto-detected if installed via Steam)
   - repak.exe and texconv.exe (included in this download, auto-detected)
4. Click a genre → "Add movie" → upload your cover image.
5. Click "Ship to Store" to generate and install the mod.
6. Launch Retro Rewind — your custom covers appear in-game!

FEATURES
--------
• Replace VHS cover art for any genre (Action, Horror, Drama, etc.)
• Create New Release standees for all genres (except Adult)
• Per-layout preview that matches in-game rendering
• Custom movie titles, star ratings, and rarity settings
• One-click Ship to Store — no manual file editing needed

TROUBLESHOOTING
---------------
• Windows Defender may flag the .exe as suspicious.
  This is a false positive common with PyInstaller-built apps.
  Click "More info" → "Run anyway" to proceed.

• If the tool can't find your game, browse to:
  [Steam Library]\steamapps\common\RetroRewind\

• If textures appear black in-game, try rebuilding the pak
  (delete the Output folder first, then Ship to Store again).

• If you get an error code (e.g. E001), note it for bug reports.

• For bug reports or questions, visit the mod page on Nexus Mods.
